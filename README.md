# Regression_on_dataset_hemologlobin

# Предсказание уровня гемоглобина по изображениям ногтей

## Описание
Блокнот посвящён задаче предсказания уровня гемоглобина по фотографиям ногтей с использованием классического машинного обучения и deep learning.  
Сравниваются два подхода:
- ручное извлечение признаков + линейная модель;
- transfer learning на базе ResNet18.

## Данные
Используется датасет **Yakimov et al., 2024** с фотографиями рук и измеренным уровнем гемоглобина.

В данных есть:
- фотографии кисти;
- координаты областей ногтей и кожи;
- целевая переменная `HB_LEVEL_GperL`.

Характеристики датасета:
- **250 пациентов**;
- диапазон Hb: **44–169 г/л**;
- для каждого пациента доступны изображения и разметка областей ногтей.

## Что сделано в блокноте
- загрузка и первичный анализ датасета;
- визуализация изображений и bounding boxes;
- анализ связи цвета ногтей с уровнем Hb;
- извлечение признаков из изображений;
- балансировка обучающей выборки;
- обучение baseline-модели ElasticNet;
- обучение модели ResNet18 в режиме frozen transfer learning;
- обучение ResNet18 с полной разморозкой;
- сравнение моделей по метрикам регрессии;
- дополнительная бинаризация задачи в детекцию анемии.

## Признаки для классического ML
Для baseline-модели из изображения ногтя извлекаются числовые признаки:
- перцентили интенсивности по каналам `R`, `G`, `B`;
- центральная часть ногтя используется как наиболее информативная область.

Это даёт **42 признака** на объект.

## Модели

### 1. ElasticNet
Линейная регрессия с регуляризацией как базовая модель.

### 2. ResNet18 frozen
Предобученная `ResNet18`, где сверточная часть заморожена, а обучается только регрессионная голова.

### 3. ResNet18 full fine-tuning
Предобученная `ResNet18`, в которой дообучаются все слои модели.

## Подготовка данных
Используются два сценария:
- для ML и frozen ResNet18: сбалансированная обучающая подвыборка **100** объектов и тест **150** объектов;
- для full fine-tuning: стратифицированный split **70/15/15** по квантилям Hb.

Для deep learning применяются:
- кроп ногтя среднего пальца;
- `Resize(224x224)`;
- аугментации;
- `DataLoader`.

## Метрики
Основная метрика:
- **RMSE** — средняя ошибка предсказания уровня Hb в г/л.

Дополнительно анализируются:
- MAE;
- медиана абсолютной ошибки;
- bias.

## Результаты

### ElasticNet
- CV RMSE: **23.9 г/л**
- validation RMSE: **24.0 г/л**
- test RMSE: **20.3 г/л**

### ResNet18 frozen
- best validation RMSE: **37.6 г/л**
- test RMSE: **30.2 г/л**

### ResNet18 full fine-tuning
- best validation RMSE: **24.4 г/л**
- test RMSE: **26.4 г/л**

### Сравнение моделей
| Модель | Test RMSE, г/л | MAE, г/л |
|---|---:|---:|
| ElasticNet | 20.3 | 16.2 |
| ResNet18 frozen | 30.2 | 25.2 |
| ResNet18 fine-tuned | 26.4 | 20.2 |

## Вывод
На этом датасете **классический подход с ручными признаками и ElasticNet оказался лучше**, чем обе версии ResNet18 по тестовому RMSE.  
Полная разморозка ResNet18 улучшает результат по сравнению с frozen transfer learning, но всё равно уступает baseline.  
Основной вывод блокнота: при небольшом объёме данных и хороших ручных признаках классический ML может быть эффективнее deep learning.
